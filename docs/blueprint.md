# **App Name**: InvoiceFlow

## Core Features:

- Data Entry: Client and Item Details Input: Allow users to input client information (name, address, contact details) and item details (description, quantity, price).
- Logo Upload: Company Logo Upload: Enable users to upload their company logo to be included on the quotations and invoices.
- Document Generation: Quotation and Invoice Generation: Generate customizable quotations and invoices based on the input data.
- PDF Export: PDF Download: Allow users to download the generated quotations and invoices in PDF format.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to evoke trust and professionalism.
- Background color: Almost white (#F0F8FF) to provide a clean, neutral backdrop.
- Accent color: Darker blue (#4682B4) for key interactive elements, to ensure visibility.
- Body and headline font: 'Inter', a sans-serif font for a modern, machined, objective look.
- Use minimalist, line-based icons for clarity and a professional aesthetic.
- Employ a clean and structured layout to organize content effectively, ensure ease of navigation and improve user experience.
